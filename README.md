### Hoe Jupyter te installeren.

Om de `.ipynb` bestanden te gebruiken moet er [Jupyter](https://jupyter.org/) geinstalleerd worden.  
Gelukkig is dit makkelijk, het is namenlijk te installeren via [`pip`](https://pypi.org/project/pip/).  
`pip` komt met Python als deze wordt geinstalleerd.  

1. Open de commandline.  
2. Typ het volgende commmando om jupyter te installeren:  
```
python -m pip install jupyterlab
```
3. Wacht todat de installatie is afgerond.  
4. Typ het volgende commmando om Jupyter te starten:  
```
python -m jupyterlab
```
5. Als Jupyter voor het eerst start kan Windows vragen welke browser je wil gebruiken voor deze applicatie.  
6. Nu is Jupyter geinstalleerd en kan de `.ipynb` bestanden via Jupyter in de browser gebruikt worden.  


### Google Colab

Is Python niet beschikbaar, maar wel een Google account?  
Dan kan je via Google Colab ook deze `.ipynb` files inzien en gebruiken.  
Gebruik hiervoor de volgende link:  
> [https://colab.research.google.com/github/my-github-alt/Python_tips_and_tricks/](https://colab.research.google.com/github/my-github-alt/Python_tips_and_tricks/)  

In de Colab omgeving heeft elke cel een play-button, deze komt tevoorschijn als de muis over de cel beweegt.  
Om de code te gebruiken via Google Colab moet er ingelogd worden met een Google account.
